PC CONTROLS GUIDE

PC 常用操作指南

涵盖鼠标操作、键盘快捷键、单位编组、相机控制与界面操作的完整参考手册。 建议新手在游戏前熟悉这些基础操作，掌握操作是发挥战术的前提。

🖱 鼠标操作 📷 相机控制 🎯 单位选择 📋 命令下达 🔢 编组管理 ⌨ 功能快捷键 🖥 界面操作 💡 新手提示
🖱
鼠标操作
MOUSE CONTROLS
左键单击
选中单个单位
点击地图上的我方单位将其选中，取消之前的所有选择
左键框选
框选多个单位
按住左键拖动，框住区域内的所有我方单位将被同时选中
右键单击
移动 / 攻击
右键点击空地为选中单位下达移动命令；右键点击敌方单位下达攻击命令
右键长按拖动
设置移动方向
移动到目标点时，通过拖动指定单位抵达后的朝向，便于建立防御阵线
滚轮滚动
缩放地图
向上滚动放大视图，向下滚动缩小视图，快速切换战术与战略视野
中键按下拖动
平移相机
按住鼠标中键并拖动，快速移动到地图任意位置，比键盘移动更精准
左键双击单位
选中同类型全部单位
双击一个单位，会自动选中当前屏幕内所有相同类型的己方单位
左键双击空地
取消所有选择
快速取消当前选中状态，避免误操作
📷
相机控制
CAMERA CONTROLS
W A S D
移动相机
经典WASD方式移动相机视角，上下左右平移
↑ ← ↓ →
方向键移动相机
方向键同样可以移动相机，与WASD效果相同
鼠标移至屏幕边缘
边缘滚动相机
将鼠标移到屏幕边框附近，相机会自动向该方向平移（需在设置中开启）
+ / -
键盘缩放
使用键盘加减键进行精细的视图缩放控制
Backspace
相机回到基地
快速将相机视角拉回至己方初始部署区域
Space
相机跳转至选中单位
按空格键，相机立即跳转至当前选中单位的位置，快速追踪关键部队
🎯
单位选择
UNIT SELECTION
Ctrl + 左键
追加/移除单位
在当前选中基础上追加点击的单位；若该单位已在选中列表中，则将其移除
Shift + 左键框选
追加框选范围
在保留当前选择的同时，将框选区域内的新单位追加进选中列表
Ctrl + A
全选屏幕内单位
选中当前屏幕视野范围内的所有己方单位，快速集结调度
Tab
循环切换选中单位
在已选中的多个单位之间逐一切换焦点，便于检查每个单位状态
Esc
取消选择 / 取消命令
取消当前所有单位选择，或取消正在执行的待确认命令
左键单击空地
取消选择
点击无单位的空白区域，清除所有选中
📋
命令下达
ORDERS
右键 点击地面
移动命令
命令选中单位向目标位置移动，系统自动规划最短安全路径
右键 点击敌方单位
攻击命令
命令选中单位攻击目标，单位将在合适射程内停下并开火
Shift + 右键
追加路径点
为单位添加移动队列中的下一个航点，实现多点路径规划
A + 右键
攻击移动（A-Move）
边移动边自动攻击沿途遭遇的敌方单位，到达目标点前不会忽略威胁
S
停止
命令选中单位立即停止所有移动与攻击动作，保持当前位置
H
原地驻守
单位停在当前位置不主动移动，但仍会自动还击射程内的敌人
P
巡逻
在当前位置与指定目标点之间往返巡逻，遭遇敌人自动攻击
D
部署/展开
命令支援单位（如炮兵）展开并进入战斗准备状态，或命令步兵卧倒隐蔽
🔢
编组管理
GROUP MANAGEMENT
Ctrl + 1 ~ 0
创建编组
将当前选中的单位设置为对应数字编组，最多可创建10个编组（0键为第10组）
1 ~ 0
选中编组
按下数字键立即选中该编组内的所有单位，无需手动在地图上点击寻找
连按两次 1 ~ 0
相机跳转至编组
快速双击数字键，在选中编组的同时将相机视角跳转至该编组位置
Shift + Ctrl + 数字
追加到编组
将当前选中的单位追加到已存在的编组中，不会清空原有编组成员
📌 编组战术建议
  • 建议将侦察单位单独编为 1 号组，始终优先掌握战场视野
  • 主战坦克编为 2 号组，步兵战车编为 3 号组，实现装甲步兵协同
  • 防空单位单独编组，随时可以呼叫拦截突然出现的航空威胁
  • 炮兵与后勤编为最后的编组，需要时快速下达支援命令

功能快捷键
FUNCTION HOTKEYS
按键 功能 说明
F1 单位能力 1 激活选中单位的第一个特殊能力（冲刺、烟雾弹、部署等）
F2 单位能力 2 激活选中单位的第二个特殊能力
F3 单位能力 3 激活选中单位的第三个特殊能力
F4 单位能力 4 激活选中单位的第四个特殊能力
F5 单位能力 5 激活选中单位的第五个特殊能力
Q 切换开火规则 在「自由开火」「仅还击」「保持戒备」之间循环切换
E 上下车 / 搭乘 命令步兵上车搭乘选中载具，或下令车载步兵下车
R 撤退 单位以最快速度向己方出生区方向撤退，忽略沿途敌人
T 强制攻击 激活强制攻击模式，点击任意目标（包括友军）实施攻击
Y 切换攻击优先级 在攻击最近目标 / 攻击最危险目标之间切换
Z 呼叫支援单位 打开可用增援单位面板，选择并呼入增援
X 标记地图目标 在鼠标位置放置战术标记，提示队友关注该区域
Delete 解散 / 击沉单位 主动移除选中单位（主要用于清除无用后勤车辆，回收点数）
Pause / Break 暂停游戏 在单人战役或自定义游戏中暂停，多人模式下无效
+ / -（小键盘） 加速 / 减速 在单人模式中调整游戏速度，最高4倍速（多人模式无效）
M 打开/关闭迷你地图 切换右下角迷你地图的显示或隐藏
N 切换单位名称显示 在地图上显示或隐藏所有单位的名称标签

界面操作
UI OPERATIONS
Tab（聊天框外）
打开聊天框
多人游戏中向队友发送战术信息或作战指令
Enter
发送聊天
在聊天框输入完毕后按Enter发送，再按一次Enter可快速打开聊天框
点击 增援面板
呼叫增援单位
从右侧甲板面板选择并点击单位图标，在地图上点击部署位置即可呼入
点击 单位信息卡
查看单位详情
选中单位后，底部信息栏显示生命值、弹药量、移速与当前任务状态
右键 单位信息卡
快速下达命令
在底部单位信息卡上右键，弹出可用命令列表，适合精细操作
拖动 迷你地图
快速移动相机
直接在迷你地图上点击或拖动，相机立即跳转至对应战场位置
点击 弹药条
查看弹药状态
单位信息栏的弹药条变红时表示弹药不足，需优先安排后勤补给
点击 血条颜色
识别伤亡程度
血条为绿色（健康）→ 黄色（受损）→ 红色（濒死），及时撤退维修
💡
新手战术提示
BEGINNER TIPS
01
侦察先行，永远优先
在移动主力部队之前，先用侦察单位（狙击组、无人机、侦察车）探明前方地形与敌军位置。 被伏击的装甲部队往往损失惨重，而主动侦察的部队才能掌握战场主动权。 看不见的敌人比看见的更危险。
02
编组是效率的关键
建议游戏开始第一件事就是为各兵种分配编组快捷键。 一旦战斗打响，你需要在几秒内同时指挥多条战线， 没有编组的部队将很快失去控制。 建议：1侦察 / 2坦克 / 3步兵战车 / 4防空 / 5炮兵 / 6直升机。
03
防空单位不可或缺
武装直升机与攻击无人机可以在几十秒内摧毁一整支装甲编队。 无论选择何种战术风格，甲板中至少需要1-2个防空单位。 铠甲-S1或爱国者的存在将迫使对手三思再三才敢出动空中力量。
04
后勤决定战争耐力
弹药耗尽的主战坦克和空载的武装直升机毫无战斗力。 在甲板中安排至少1-2辆补给车，并规划安全的后勤补给路线。 补给车靠近友军单位时会自动开始补给，无需额外操作。
05
地形是天然护盾
树林、建筑物和山丘可以为单位提供遮蔽，隐藏于树林中的步兵几乎无法被远距离发现。 善用地形隐蔽防御部队，或利用地形掩护实施侧翼迂回， 是区分新手与老手的核心战术素养。
06
使用攻击移动代替普通移动
普通移动（右键）的单位在路途中会无视敌人径直前进，可能走进敌方埋伏。 使用攻击移动（A + 右键）可以让单位在前进过程中自动交战， 大幅降低被偷袭损失。除快速增援等紧急情况外，建议优先使用攻击移动。
07
开火规则决定生存率
默认的「自由开火」会让单位主动向远处的敌人开火，暴露自身位置。 在侦察或潜伏场景下，将单位设为「仅还击」（Q切换）， 单位只在被攻击时才会还手，大幅提升生存能力与伏击效果。
08
了解单位的最大射程
选中单位后，地图上会显示该单位的武器射程圆圈。 利用射程优势可以在敌人攻击你之前就开火。 长程反坦克导弹（科尔内特/标枪）可在坦克反制前就完成攻击， 这是步兵对抗重型装甲的关键战术。
